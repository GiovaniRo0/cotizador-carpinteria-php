Cotizador Web para Carpintería

Aplicación web para generar cotizaciones automáticas de muebles de carpintería.

Tecnologías
- HTML5
- PHP
- MySQL
- CSS3

Funcionalidades
- Cálculo automático de precios
- Almacenamiento de cotizaciones
- Interfaz amigable para clientes

Instalación
1. Clonar repositorio en `appserver`
2. Crear base de datos
3. Configurar conexión 
4. Acceder vía `http://localhost/cotizador`
